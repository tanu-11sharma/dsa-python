## Structure

Each file is self-contained: a docstring with the problem statement and time/space complexity, the solution, and a `__main__` block with example test cases you can run directly:

```bash
python3 two_sum.py
```

Consistent, daily DSA practice — building breadth across arrays, trees, graphs, DP, backtracking, and more, one clean solution at a time.
